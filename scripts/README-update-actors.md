# Обновление актёров для существующих фильмов

## Проблема
Старые фильмы в базе данных могут иметь только одного актёра, так как при первоначальном импорте было ограничение на количество актёров.

## Решение

### Вариант 1: Обновление через админ-панель (рекомендуется)

1. Откройте страницу редактирования фильма: `/admin/movies/[id]`
2. Прокрутите вниз до раздела "Обновление актёров"
3. Нажмите кнопку "Обновить актёров из TMDB"
4. Дождитесь завершения обновления

Этот способ обновляет актёров для одного фильма за раз.

### Вариант 2: Массовое обновление через скрипт

Запустите скрипт для обновления всех фильмов с менее чем 5 актёрами:

```bash
npx tsx scripts/update-movie-actors.ts
```

Или обновите конкретный фильм:

```bash
npx tsx scripts/update-movie-actors.ts [movieId]
```

**Примечание:** Скрипт автоматически пропускает фильмы, у которых уже есть 5+ актёров.

## Как это работает

1. Скрипт/API ищет фильм в TMDB по названию и году выпуска
2. Если фильм найден, загружаются все актёры из TMDB
3. Для каждого актёра:
   - Проверяется, существует ли актёр в базе данных
   - Если нет - создаётся новый актёр с данными из TMDB
   - Создаётся связь между фильмом и актёром

## Требования

- Должен быть установлен `TMDB_API_KEY` в переменных окружения
- Фильм должен существовать в базе данных TMDB

## Ограничения

- Скрипт может не найти фильм в TMDB, если название сильно отличается
- TMDB API имеет лимиты на количество запросов (40 запросов в 10 секунд)
