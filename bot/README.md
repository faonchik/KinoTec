# 🤖 Telegram бот КиноТека

Бот для управления контентом и каталогом сайта КиноТека напрямую через Telegram.

## 🛠 Возможности бота
- **📝 Управление контентом**: Создание новостей, интервью, обзоров и статей с помощью AI (Groq/Llama 3.3). Автоматический поиск обложек на Unsplash и TMDB.
- **🎬 Импорт фильмов**: Поиск и импорт фильмов из базы TMDB (включая постеры, обложки, актеров, режиссеров и детали).
- **📺 Импорт сериалов**: Поиск и импорт сериалов с разбивкой по сезонам и эпизодам из TMDB.
- **👥 Модерация пользователей**: Просмотр списка пользователей, топ активных, поиск.
- **⭐ Модерация отзывов**: Одобрение или отклонение отзывов на модерации.
- **📊 Аналитика сайта**: Быстрая статистика по фильмам, пользователям, просмотрам и оценкам.
- **💬 AI Помощник**: Свободный диалог с AI.

## 🚀 Настройка и запуск

### 1. Переменные окружения (.env)
Бот использует переменные окружения из файла `.env` в корне проекта. Убедитесь, что там заданы следующие параметры:

```env
# Токен бота, полученный у @BotFather
TELEGRAM_BOT_TOKEN=ваш_токен_бота

# Ваш числовой ID в Telegram (для авторизации). Можно узнать через бота @userinfobot
TELEGRAM_ADMIN_ID=ваш_числовой_id_telegram

# Ключ API Groq для работы AI-генератора новостей и статей
GROQ_API_KEY=ваш_ключ_groq

# Ключ API TMDB для импорта фильмов и сериалов
TMDB_API_KEY=ваш_ключ_tmdb

# Ключ API Unsplash для поиска картинок к генерируемым новостям
UNSPLASH_ACCESS_KEY=ваш_ключ_unsplash

# Ссылка на сайт (для генерации ссылок в сообщениях бота)
SITE_URL=http://localhost:3000
```

---

### 2. Запуск через Docker Compose (Рекомендуемый)

Бот полностью интегрирован в общую Docker-инфраструктуру проекта и запускается одной командой вместе с сайтом и базой данных:

```bash
# Запуск всего проекта (включая бота)
docker-compose up -d --build

# Перезапуск только бота при обновлении кода
docker-compose up -d --build bot

# Просмотр логов бота
docker-compose logs -f bot
```

---

### 3. Локальный запуск (для разработки)

Для запуска бота локально без Docker выполните следующие шаги:

1. Перейдите в папку бота:
   ```bash
   cd bot
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите бота в режиме разработки (автоматическая перезагрузка при изменении кода):
   ```bash
   npm run dev
   ```
4. Для сборки и запуска в production-режиме:
   ```bash
   npm run build
   npm start
   ```
